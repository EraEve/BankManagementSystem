## 📊 数据结构与算法应用

| 数据结构/算法 | 应用场景 | 复杂度 |
|-------------|---------|--------|
| **vector 动态数组** | 所有数据存储（职员/客户/卡/交易） | O(1) 随机访问 |
| **map 哈希表** | ID→Index 快速索引 | O(log N) / O(1) |
| **LinkedQueue 链式队列** | 银行排队窗口等候队列 | O(1) 入队/出队 |
| **优先队列逻辑** | VIP客户优先服务 | O(1) VIP优先判断 |
| **图(邻接矩阵) + Dijkstra** | 网点最短路径导航 | O(V²) |
| **BST 二叉搜索树** | 客户ID动态索引 | O(log N) |
| **冒泡排序** | 交易按时间排序 | O(N²) |
| **快速排序** | 交易按金额排序 | O(N log N) |
| **二分查找** | 有序客户查找 | O(log N) |
| **滑动窗口** | 高频交易异常检测 | O(N) |
| **加权多因子模型** | 信用评分 & 风控审批 | O(1) 评分计算 |
| **文件I/O** | 所有数据持久化(管道分隔符) | O(N) |

## 🎯 功能模块详情

### 需求1: 银行职员管理 ✅
- 管理员增删改查职员信息
- 文件存取职员数据 (`data/employee.txt`)
- 职员修改登录密码和个人信息
- 数据结构: `vector<Employee>` + `map<string,int>` 哈希索引

### 需求2: 客户账户管理 ✅
- 职员增删改查客户资料（含VIP/普通分类）
- 客户字段: 编号/姓名/密码/类型/电话/开户日期/信用评分/金融资产/地址/身份证
- 广义表: 一个客户关联多张银行卡
- 数据结构: `vector<Customer>` + `map<string,int>` + BST

### 需求3: 银行卡管理 ✅
- 职员增删改查银行卡
- 类型: 借记卡/储蓄卡/信用卡
- 字段: 编号/客户ID/类型/余额/借贷余额/利率/信用额度/开户日期/状态
- 数据结构: `vector<BankCard>` + `map<string,int>`

### 需求4: 存取贷业务管理 ✅
- 存款/取款/转账/贷款/还款 完整业务流程
- 利息计算（日利息/月利息）
- 余额校验/日限额校验/状态校验
- 自动更新客户金融资产
- 数据结构: `vector<Transaction>`

### 需求5: 业务查询 ✅
- 按时间段查询（起止日期）
- 按客户类型查询（VIP/普通）
- 按银行卡号查询
- 按金额区间查询
- 按交易类型查询（收入/支出）
- **组合查询**（多条件AND）
- **业务汇总统计**（各类型笔数+金额）
- 排序: 冒泡排序(时间) + 快速排序(金额)

### 需求6: 银行排队管理 ✅
- VIP窗口 + 普通窗口 双队列管理
- 客户取号（自动识别VIP/普通）
- VIP优先叫号策略
- 实时排队状态显示
- 客户业务评分（1-5分）
- 每日客流统计
- **数据结构核心: LinkedQueue链式队列**

### 需求7: 网点查询与导航 ✅
- 网点地图（5个内置网点 + 可扩展）
- 网点增删改查
- 网点间路径设置
- **Dijkstra最短路径导航**（含路径重构）
- 查询所有可达网点及距离
- **数据结构核心: 图(邻接矩阵) + Dijkstra算法**

### 需求8: 智能管理（创新功能）✅
1. **交易异常报警**
   - 大额交易检测 (>50000元)
   - 高频交易检测 (滑动窗口, 1小时内>5笔)
2. **批量利息计算**
   - 全行每日利息汇总
   - 全行每月利息汇总
3. **信用评级 & 风控审批**
   - 加权多因子信用评分模型
   - AAA/AA/A/B/C 五级评级
   - 风控审批（负债率/资产/信用综合评估）
4. **投资顾问**
   - 基于信用评分的风险偏好判断
   - 进取型/稳健型/保守型三档资产配置建议

### 需求9: 主界面 ✅
- 字符图形界面（BOX-DRAWING字符边框）
- 三级菜单体系: 主界面 → 角色菜单 → 功能菜单
- 统一登录: 管理员/职员/客户三种身份
- 新客户自助注册

### 需求10: 设计报告 ✅
- 本README + 完整源代码 + 中期报告.docx

## 🔧 编译与运行

### 环境要求
- C++11 及以上 (GCC 4.8+ / MinGW / Visual Studio 2015+)
- Windows / Linux / macOS

### 编译

```bash
# 方式1: 直接编译
g++ main.cpp -o bank_system -std=c++11

# 方式2: 使用Makefile (Linux/macOS/MinGW)
make

# 方式3: Visual Studio
# 新建C++控制台项目, 将所有.h和.cpp文件添加到项目中编译
```

### 运行

```bash
# Windows
bank_system.exe

# Linux/macOS
./bank_system
```
